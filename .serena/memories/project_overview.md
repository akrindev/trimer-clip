Project: trimer-clip
Purpose: AI-powered video editing pipeline to turn long-form videos into short clips (TikTok/Shorts/Reels) with transcription, highlight detection, trimming, portrait resize, and subtitle burn-in.
Core stack: Python 3.9+, FFmpeg, Whisper/faster-whisper, pyannote, OpenCV, MoviePy, Torch. Optional Gemini API for transcription/analysis.
