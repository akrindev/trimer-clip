Key structure:
- skills/: modular CLI skills (youtube-downloader, video-transcriber, scene-detector, highlight-scanner, video-trimmer, portrait-resizer, subtitle-overlay, autocut-shorts, batch-processor, etc.)
- scripts/shared/: shared utilities (ffmpeg_wrapper, audio_utils, video_utils, subtitle_utils)
- scripts/models/: transcription models (whisper_transcriber, gemini_transcriber)
- fonts/: bundled fonts (Plus Jakarta Sans)
- downloads/: sample inputs/outputs
- references/: docs and ffmpeg notes
